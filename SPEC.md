# OpenCRO Homepage Spec
## Version 1.0 — Single Page Conversion Architecture

---

# 1. OBJECTIVE

Create a **high-converting single-page homepage** that:

- Establishes authority in FDA cybersecurity
- Creates passive FOMO (status, control, clarity)
- Creates active FOMO (limited capacity)
- Drives users into **2 decisions only**:
  - Clear ($43K)
  - Model ($6K)

---

# 2. CORE PRINCIPLES

- Outcome > explanation
- Decision > exploration
- Constraint > abundance
- Precision > persuasion

---

# 3. USER FLOW

## Primary Flow

LinkedIn / DM → Homepage (scroll) → CTA click → /clear → Calendly → Deal

## Secondary Flow

Homepage → Model → /model → Upsell to /clear → Deal

---

# 4. NAVIGATION (Sticky Header)

Left:
- Logo (OpenCRO)

Center (anchor links):
- How it works (#model)
- Get FDA-ready (#buy)
- Start with a model (#model-entry)
- About (#about)

Right (primary CTA):
- Get FDA-Ready – $43K → /clear

## Behavior

- Sticky
- Transparent at top
- Solid background on scroll
- Smooth scroll to anchors

- Background: #0B0B0B
- Text: #F5F5F5

- ALL CAPS
- Letter spacing: +10%
- No boxes

Highlight "EXPLOITABILITY" with accent color

- Multiple offers in decision sections
- Stock images
- Cybersecurity clichés
- Complex navigation
- "Learn more" loops
